## Supermarket Sales Analysis

End-to-end sales analysis on a Nigerian supermarket dataset using Python and pandas.

**Tools:** Python, pandas, Matplotlib, Seaborn
**Skills demonstrated:** Data cleaning, EDA, groupby analysis, time series, data visualisation

**Key findings:**
- Lagos generated the highest revenue across all three cities
- Electronics led in quantity sold but Clothing led in revenue
- Monthly revenue showed volatility with peaks in August and December
